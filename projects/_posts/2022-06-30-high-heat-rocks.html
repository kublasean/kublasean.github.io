Using the same tech stack as this blog I created a custom website for my band, High Heat. Check it out here! Some features are:

  • Slick animated menu dropdown on small screens
  • Faster image load times with srcset elements
  • Responsive art direction for hero image on home page for horizontal or vertical screens with picture element
  • Photo gallery with small, compressed thumbnails to reduce page load
  • Responsive font size scaling
  • Flame emoji favicon (best feature if you ask me!!)
